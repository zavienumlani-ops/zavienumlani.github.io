Study Tips

Groups (Columns) Elements in the same group have similar properties and the same number of valence electrons.
Periods (Rows) Elements in the same period have the same number of electron shells.
Atomic Number The number of protons in an atom's nucleus. This determines the element.
Atomic Mass The average mass of an element's atoms, measured in atomic mass units (amu).
